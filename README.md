# Customer Segmentation Using K-Means Clustering

This project implements customer segmentation using the K-Means clustering algorithm to group customers based on their features and behavior patterns.

## Overview

Customer segmentation is a common unsupervised learning task used to identify distinct groups within a customer base.  
This project uses K-Means clustering to:
- Analyze customer data
- Discover natural groupings among customers
- Visualize and interpret the resulting clusters

## Files

- 'project.ipynb' — Main notebook containing the implementation
- 'Customer_Segmentation_Data.csv' — File containing the dataset
- 'segmented_customers.csv' — File containing segmented customers after implementing
- 'requirements.txt' — Python dependencies
- 'screenshots/' —  Project screenshots or output figures

## Requirements

Python 3.8+ and the packages listed in 'requirements.txt'.

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Open project.ipynb in Jupyter Notebook
3. Run the cells from top to bottom.
